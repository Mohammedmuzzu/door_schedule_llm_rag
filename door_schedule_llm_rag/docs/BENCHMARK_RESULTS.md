## Door Schema Fill Analysis
The 61.2% average fill is expected because many architectural PDFs only include a subset of all possible fields. Common missing fields:
- `door_thickness` — rarely specified in schedules
- `door_finish` / `frame_finish` — often lumped into notes
- `elevation` — only in certain project types
